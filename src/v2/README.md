# Simple RPC v2

简单的RPC实现例子，通过这个例子你可以理解一个基本的RPC框架的实现原理。

- `RpcClient` : RPC 客户端，连接服务器，将方法名和参数发送给服务器，并接收服务器放回的结果。
- `RpcRequet` : RPC 服务端：接收客户端请求，找到对应的方法并调用，将结果返回给客户端。
- `RpdServer` : RPC 请求类：用于封装客户端发送的请求数据，包括方法名和参数。
